# plg_fbsocialog
Add metatags :og to the articles content

For share a article in Facebook need add metatags og with title, description, image and type of content, with this plugin this is automatic, take this info for the article info.

Is better if add the App ID of facebook in the plugin configuration.

If need check how facebook have the info about your web go to https://developers.facebook.com/tools/debug/sharing/



## Spanish info
Agrega las metatags OG de Open Graph de Facebook a nuestro sitio

En todo el sitio agrega el App ID y el Facebook Page ID
En nuestro componente de Articulos (content) toma los valores del titulo, imagen, descripcion, etc. del Articulo y genera las tags necesarias.

Si requieres verificar como Facebook tiene los datos de tu pagina visita https://developers.facebook.com/tools/debug/sharing/
